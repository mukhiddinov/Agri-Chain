How a transaction flows

Order created

Order API receives intake, persists intent, and emits order.created to Kafka.

Inventory reserved

Orchestrator invokes inventory gRPC to reserve quantities and validate supply buffers.

Escrow funded

Payment service triggers escrow funding on Sepolia, returning verifiable transaction hashes.

Settlement & release

Delivery or exception events unlock funds, or trigger compensation through saga steps.
